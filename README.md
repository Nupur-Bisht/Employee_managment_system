# Employee_managment_system
A simple Employee Management System built using Core Java and MySQL.
This project allows you to add, delete, and view employee details with a user-friendly interface.
Features
1. Add new employees
2. Delete employees
3. Search employees
4. View all employees
5. Data stored in MySQL database
6. Secure database connection using JDBC
